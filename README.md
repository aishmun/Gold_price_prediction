# Gold_price_prediction

**Gold Price Prediction Using Machine Learning & Deep Learning**

A predictive analytics system developed to forecast gold prices using Machine Learning and Deep Learning models. The project focuses on analyzing historical financial data and generating accurate predictions through data-driven modeling techniques.

**🚀 Project Overview**

_This project integrates data analysis, preprocessing, visualization, and predictive modeling to forecast gold price trends. Multiple ML/DL approaches were explored to analyze patterns in financial time-series data and evaluate forecasting performance._

**🛠️ Technologies Used**

1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Scikit-learn
6. TensorFlow & Keras
7. SQL
8. FastAPI

**✨ Key Features**

1. Historical gold price data analysis
2. Time-series forecasting using ML/DL models
3. Data preprocessing and visualization pipeline
4. Comparative model evaluation
5. Interactive prediction and visualization interface
6. Scalable backend integration using FastAPI


**👩‍💻 My Contributions**

1. Performed data collection using financial datasets and APIs for historical gold price analysis.
2. Conducted data preprocessing, including handling missing values, normalization, and feature preparation for model training.
3. Performed exploratory data analysis (EDA) to identify market trends, correlations, and price movement patterns through visualizations.
4. Worked on model training and deployment for real-time prediction functionality.
5. Conducted model evaluation using performance metrics such as Mean Squared Error (MSE) and accuracy comparison techniques.
6. Assisted in building an interactive system for prediction visualization and user interaction.

**📊 Outcome**
The system successfully analyzed historical gold price trends and generated predictive forecasts using machine learning and deep learning approaches, helping improve understanding of financial market behavior.
